# Reports

This folder stores JSON evaluation reports generated by the pipeline.

## What Goes Here

- run timestamp
- question used for the run
- generated answer
- retrieval summary
- evaluation scores

## Purpose

These reports give us a simple history of pipeline runs so we can compare:

- baseline vs improved retrieval
- answer quality over time
- evaluation score changes across experiments

## Notes

- Files in this folder are generated automatically.
- Each report is saved as a timestamped JSON file.
